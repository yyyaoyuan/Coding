# 冒泡排序

首先遍历一遍数组，比较两个相邻的元素，如果前面的元素大于后面的元素，便交换位置，此时遍历一轮，最大的元素便会出现在最后一个位置；
然后按照相同的方式第二次遍历数组，但是此时不遍历最后一个元素，此时次大的元素会出现在倒数第二个位置；
依次进行，直至所有的元素排序完成。

```python
def popSort(A):
    for i in range(len(A), 0, -1):
        for j in range(i-1):
            if A[j] > A[j+1]:
                A[j], A[j+1] = A[j+1], A[j]
    return A


A = [2,4,1,2,5,3,7,2]
print(popSort(A))
```
